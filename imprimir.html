4 — Revisão Express: Linguagens
Português — Domínio da Língua
Figuras de linguagem: metáfora, metonímia, ironia, hipérbole, eufemismo, antítese, paradoxo
Funções da linguagem: referencial, poética, emotiva, fática, metalinguística, conativa
Coesão: referência, substituição, conjunção, elipse
Concordância verbal: sujeito coletivo, pronomes "SE", orações infin.
Regência: assistir A, obedecer A, namorar, preferir A…
Crase: antes de fem. determinado; não usa com verbos, masc., pronomes
Pontuação: vírgula nunca separa sujeito de predicado
Literatura Brasileira
Romantismo (1836–1880): subjetivismo, idealização, indianismo (Alencar: "Iracema"), ultrarromantismo (Álvares de Azevedo)
Realismo/Naturalismo (1881–1902): objetividade, crítica social. Machado de Assis: "Dom Casmurro", "Memórias Póstumas". Aluísio Azevedo: "O Cortiço"
Parnasianismo: "arte pela arte", Olavo Bilac
Modernismo 1ª fase (1922): Semana Arte Moderna. Oswald, Mário de Andrade. Ruptura com normas
Modernismo 2ª fase (1930–45): engajamento social. Graciliano Ramos: "Vidas Secas". Jorge Amado: "Capitães da Areia"
Modernismo 3ª fase (1945+): Guimarães Rosa: "Grande Sertão: Veredas"; Clarice Lispector
Inglês — Essentials
Simple Present: habits, facts (he plays)
Present Continuous: now, temporary (is playing)
Simple Past: finished actions (played / irregular: went, saw)
Future: will (spontaneous) / going to (planned)
Present Perfect: past with present result (have/has + pp)
Pronomes: subject (I/you/he) · object (me/you/him) · possessivo (my/your/his)
False friends: actually = na verdade; sensible = sensato; library = biblioteca
Interpretação de Texto
Leia o enunciado ANTES do texto
Inferência: resposta está implícita no texto, não na sua opinião
Pressupostos e subentendidos: o que o texto assume como verdade
Tipologia: narrativo, descritivo, dissertativo, injuntivo, expositivo
Gêneros: charge, crônica, editorial, conto, poema, propaganda
Ironia: dizer o contrário do que se quer significar

6 — Revisão Express: Matemática
Aritmética e Porcentagem
Razão: a/b. Proporção: a/b = c/d → ad = bc
Regra de 3 simples: montar a proporção, cruzar
Regra de 3 composta: montar tabela, verificar se grandezas são diretas ou inversas
Porcentagem: x% de N = (x/100)·N. Aumento: ×(1 + x/100). Desconto: ×(1 - x/100)
Juros simples: J = C·i·t. Montante: M = C(1 + i·t)
Juros compostos: M = C·(1+i)^t
MMC / MDC: MMC para somar frações; MDC para simplificar
Funções
1º grau: f(x) = ax + b. Raiz: x = -b/a. Gráfico: reta
2º grau: f(x) = ax² + bx + c. Δ = b² - 4ac
Raízes: x = (-b ± √Δ) / 2a. Δ>0: 2 raízes; Δ=0: 1; Δ<0: nenhuma
Vértice: x_v = -b/2a ; y_v = -Δ/4a
Sinal: a>0 → parábola abre pra cima (mínimo); a<0 → pra baixo (máximo)
Função exponencial: f(x)=a^x. Logaritmo: log_a(b) = x ↔ a^x = b
Geometria Plana
Triângulo: A = base·altura/2. Pitágoras: a² = b² + c²
Quadrado: A = L². Retângulo: A = b·h. Losango: A = D·d/2
Círculo: A = π·r². Circunferência: C = 2π·r
Trapézio: A = (B+b)·h/2
Triângulos notáveis: 30-60-90 (lados 1, √3, 2) · 45-45-90 (lados 1, 1, √2)
Volumes: cubo V=a³ · paralelepípedo V=c·l·a · cilindro V=π·r²·h · cone V=π·r²·h/3
Probabilidade e Estatística
P(A) = casos favoráveis / casos totais. 0 ≤ P ≤ 1
Eventos independentes: P(A∩B) = P(A)·P(B)
Eventos mutuamente exclusivos: P(A∪B) = P(A) + P(B)
Média aritmética: x̄ = (Σxi) / n
Mediana: valor central (dados ordenados)
Moda: valor mais frequente
Permutação: P_n = n! Combinação: C(n,k) = n! / (k!·(n-k)!)
7 — Revisão Express: Natureza
Biologia — Célula
Procariótica: sem núcleo (bactérias, arqueias). Eucariótica: com núcleo (animais, plantas, fungos)
Organelas: mitocôndria (ATP, respiração), cloroplasto (fotossíntese), ribossomo (síntese proteica), RE rugoso (proteínas), RE liso (lipídios), Golgi (embalagem e secreção)
Fotossíntese: 6CO₂ + 6H₂O + luz → C₆H₁₂O₆ + 6O₂. Ocorre no cloroplasto
Respiração aeróbica: glicose + O₂ → CO₂ + H₂O + ATP. Mitocôndria
Divisão: Mitose (2 células = mãe) · Meiose (4 células com metade dos cromossomos — gametas)
DNA: dupla hélice, AT e CG. RNA: mensageiro, ribossomal, transportador
Biologia — Ecologia
Cadeia alimentar: produtor → consumidor primário → secundário → terciário → decompositores
Biomas brasileiros: Amazônia (maior), Cerrado (maior biodiversidade relativa), Caatinga (único exclusivo do Brasil), Mata Atlântica (mais ameaçada: 12% restante)
Ciclos biogeoquímicos: carbono (combustíveis, fotossíntese), nitrogênio (fixação por bactérias), água
Relações ecológicas: mutualismo (+/+), comensalismo (+/0), parasitismo (+/-), predatismo (+/-)
Evolução: seleção natural (Darwin), mutação, deriva genética, especiação
Química
Tabela periódica: períodos (linhas) = níveis de energia · grupos (colunas) = valência
Metais (esq): condutores, maleáveis. Não-metais (dir): maus condutores. Gases nobres: inertes
Ligações: iônica (metal+não-metal, transfere e-) · covalente (não-metal+não-metal, compartilha e-) · metálica
Reações: síntese (A+B→AB) · decomposição (AB→A+B) · deslocamento · dupla troca
Ácido: libera H⁺. Base: libera OH⁻. Sal: ácido + base. pH <7 ácido; >7 básico; =7 neutro
Org: alcanos (CₙH₂ₙ₊₂), alquenos (C=C), aromáticos (benzeno). IUPAC: prefixo + infixo + sufixo
Física
Cinemática: v = Δs/Δt · a = Δv/Δt · v² = v₀² + 2aΔs · s = s₀ + v₀t + at²/2
Dinâmica: F = ma (2ª Lei Newton) · ação e reação (3ª Lei) · P = mg (g ≈ 10 m/s²)
Trabalho: W = F·d·cos θ. Potência: P = W/t. Energia cinética: Ec = mv²/2
Energia potencial gravitacional: Epg = mgh
Termodinâmica: calor flui do quente para o frio. Q = m·c·ΔT (calor sensível)
Ondas: v = λ·f. Som ≈ 340 m/s no ar. Luz ≈ 3×10⁸ m/s
Eletricidade: V = R·I (Lei Ohm) · P = V·I · série: R_total = R₁+R₂ · paralelo: 1/R_t = 1/R₁+1/R₂
