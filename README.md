## Como usar o ciclo diário ideal (4h)

**Pomodoro 25/5** — ou seja, 25 min foco + 5 min pausa, repetido 6× = ~3h foco + 30 min pausa = 4h "puro suor".

```
Bloco 1 (1h) → Material novo (resumo + flashcard de uma carta)
Bloco 2 (1h) → Quiz / exercícios do tópico do bloco 1
Bloco 3 (1h) → Revisão espaçada (flashcards de dias anteriores)
Bloco 4 (1h) → Treino direcionado (redação OU prova antiga)
```

Termine cada dia atualizando o `progresso.json` no dashboard com o que estudou e como sentiu (1-5).

## Regras de ouro (resumo do PLANO.md)

1. **4h reais > 6h olhando o livro.** Use pomodoro.
2. **Nunca pule revisão espaçada.** O sistema é eficaz porque revisita.
3. **Redação 1× por semana mínimo.** Sem treino = sem nota.
4. **1 prova anterior cronometrada por semana.** Não fragmentada.
5. **Dia anterior à prova: zero conteúdo novo.** Só descanso + revisão leve.
6. **Se faltar tempo, corta Natureza/Humanas — nunca Redação ou Linguagens.**
