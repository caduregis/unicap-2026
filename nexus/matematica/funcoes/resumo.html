1
Conceitos Fundamentais

Uma função f de A em B é uma relação que associa a cada elemento de A um único elemento de B. A função é uma das ideias mais importantes da matemática por modelar dependência entre grandezas.

Domínio

Conjunto A — todos os valores de entrada possíveis. Para f(x)=√x, o domínio é x≥0. Para f(x)=1/x, domínio é ℝ−{0}.

Imagem

Conjunto de todos os valores que f(x) efetivamente assume. Pode ser menor que o contradomínio. Ex: f(x)=x² tem imagem ℝ≥0 mesmo com domínio ℝ.

Contradomínio

Conjunto B, declarado como possível destino. A imagem está contida no contradomínio. Quando Im=CD, a função é sobrejetora.

Injetora, Sobrejetora, Bijetora

Injetora: elementos distintos têm imagens distintas. Sobrejetora: Im=CD. Bijetora: injetora E sobrejetora — permite definir a inversa f⁻¹.

f: A → B é função ⟺ ∀x ∈ A, ∃! y ∈ B tal que f(x) = y Domínio natural: maior conjunto onde f faz sentido
Dica UNICAP: Questões pedem para determinar o domínio de funções compostas como f(x) = √(x−3) + 1/(x−5). Resolva cada restrição separada e intersecte.
2
Função Linear e Afim (1º Grau)

A função afim tem a forma f(x) = ax + b, onde a é o coeficiente angular (taxa de variação) e b é o coeficiente linear (intersecção com o eixo y).

Crescimento

Se a > 0: função crescente. Se a < 0: função decrescente. Se a = 0: constante (f(x) = b).

Zero da Função

Zero ou raiz: valor de x onde f(x) = 0. Para ax+b=0: x = −b/a. É o ponto onde o gráfico cruza o eixo x.

Gráfico

Sempre uma reta. Dois pontos determinam a reta. O coeficiente angular a = tan(θ), ângulo que a reta faz com o eixo x.

Função Linear Pura

Caso especial onde b = 0: f(x) = ax. Passa pela origem. Proporcionalidade direta entre y e x.

f(x) = ax + b Zero: x₀ = −b/a Coeficiente angular entre dois pontos: a = (y₂−y₁)/(x₂−x₁) Gráfico: reta com inclinação a e intersecção y = b
3
Função Quadrática (2º Grau)

A função quadrática f(x) = ax² + bx + c (a ≠ 0) tem como gráfico uma parábola. O sinal de a determina se abre para cima ou para baixo.

Vértice

Ponto máximo ou mínimo da parábola. Coordenadas: xᵥ = −b/(2a); yᵥ = −Δ/(4a) ou yᵥ = f(xᵥ). Se a>0, vértice é mínimo. Se a<0, vértice é máximo.

Discriminante Δ

Δ = b² − 4ac. Se Δ>0: duas raízes reais distintas. Se Δ=0: raiz dupla (uma raiz real). Se Δ<0: sem raízes reais (parábola não toca o eixo x).

Bhaskara

Fórmula para encontrar raízes: x = (−b ± √Δ) / (2a). Exige Δ ≥ 0 para raízes reais. As duas raízes somam −b/a e multiplicam c/a (Relações de Girard).

Forma Fatorada

Se Δ ≥ 0: f(x) = a(x − x₁)(x − x₂). Útil para analisar sinal: quando a função é positiva ou negativa. Sinal de a nas extremidades.

f(x) = ax² + bx + c (a ≠ 0) Δ = b² − 4ac x = (−b ± √Δ) / (2a) Vértice: xᵥ = −b/(2a), yᵥ = −Δ/(4a) Soma das raízes: x₁+x₂ = −b/a Produto das raízes: x₁·x₂ = c/a
Dica UNICAP: Questões frequentes: (1) encontrar vértice e raízes; (2) montar equação dado que passa por três pontos; (3) analisar para quais valores de x a função é negativa. Domine o sinal da parábola.
4
Função Exponencial e Logarítmica

A função exponencial f(x) = aᵇˣ modela crescimento e decaimento. A logarítmica é sua inversa. São funções mutuamente inversas quando mesma base.

Exponencial f(x) = bˣ

b > 0, b ≠ 1. Se b > 1: crescente, explode para +∞. Se 0 < b < 1: decrescente, tende a 0. Sempre passa por (0,1). Assíntota horizontal y = 0.

Logarítmica f(x) = log_b(x)

Inversa da exponencial. Domínio: x > 0. Se b > 1: crescente. Se 0 < b < 1: decrescente. Passa por (1,0) e (b,1). Assíntota vertical x = 0.

Propriedades do Logaritmo

log(AB) = log A + log B. log(A/B) = log A − log B. log(Aⁿ) = n·log A. Mudança de base: log_b(a) = log(a)/log(b).

Base e e Base 10

ln = log_e (logaritmo natural). log = log₁₀ (logaritmo decimal). ln e = 1; ln 1 = 0. log 10 = 1; log 100 = 2; log 0,1 = −1.

Exponencial: f(x) = a · bˣ → f(x) > 0 para todo x Logarítmica: f(x) = log_b(x) → definida só para x > 0 log_b(b) = 1 log_b(1) = 0 log_b(bˣ) = x b^(log_b(x)) = x (inversa) Propriedades: log(A·B) = logA + logB log(A/B) = logA − logB log(Aⁿ) = n·logA Mudança de base: log_b(a) = log(a)/log(b)
Dica UNICAP: Equações exponenciais: iguale as bases. Ex: 4ˣ = 8 → 2²ˣ = 2³ → 2x = 3 → x = 3/2. Logaritmos aparecem em pH = −log[H⁺] e em problemas de decaimento radioativo.
5
Composição e Função Inversa

A composição aplica funções em sequência. A inversa "desfaz" a operação original, retornando ao valor inicial.

Composição (f∘g)(x)

Significa f(g(x)): aplica g primeiro, depois f ao resultado. Ordem importa: f∘g ≠ g∘f em geral. Leia "f de g de x".

Como Calcular (f∘g)(x)

Substitua o x de f pelo g(x) inteiro. Se f(x)=x²+1 e g(x)=2x−3, então f(g(x)) = (2x−3)²+1 = 4x²−12x+9+1 = 4x²−12x+10.

Função Inversa f⁻¹

Só existe para funções bijetoras. Para encontrar: escreva y = f(x), troque x↔y, isole y — esse y é f⁻¹(x). Verificação: f(f⁻¹(x)) = x.

Gráfico da Inversa

O gráfico de f⁻¹ é o reflexo do gráfico de f em relação à bissetriz y = x. Se (a,b) está em f, então (b,a) está em f⁻¹.

Composição: (f∘g)(x) = f(g(x)) Domínio de f∘g: {x ∈ Dom(g) : g(x) ∈ Dom(f)} Inversa: f⁻¹(x) tal que f(f⁻¹(x)) = x e f⁻¹(f(x)) = x Exemplo: f(x) = 3x + 2 → y = 3x+2 → x = 3y+2 → y = (x−2)/3 Portanto: f⁻¹(x) = (x−2)/3
6
Gráficos e Transformações

Translação Vertical

g(x) = f(x) + k: sobe k unidades se k > 0; desce se k < 0. Mantém a forma do gráfico, move verticalmente.

Translação Horizontal

g(x) = f(x − h): desloca h unidades para a direita se h > 0; para a esquerda se h < 0. Atenção: sinal contrário ao esperado.

Reflexões

−f(x): reflete em relação ao eixo x (inverte y). f(−x): reflete em relação ao eixo y (inverte x). −f(−x): rotação de 180° em torno da origem.

Esticamento / Compressão

a·f(x): multiplica por a na direção y. Se |a|>1: estica. Se 0<|a|<1: comprime. f(bx): afeta na direção x (b>1 comprime, 0<b<1 estica).

g(x) = a · f(b(x − h)) + k a = amplitude vertical (estica/comprime em y) b = compressão horizontal (período = T/b para periódicas) h = deslocamento horizontal (para direita se h > 0) k = deslocamento vertical (para cima se k > 0)
Dica UNICAP: Dado o gráfico de y = f(x), identificar o de y = −f(x+2) − 1: reflete no eixo x, desloca 2 unidades para a esquerda e 1 unidade abaixo. Analise cada transformação separadamente.

Questões-Tipo UNICAP — Resolvidas
Questão 1 — Função 2º Grau
A função f(x) = x² − 6x + 8 possui raízes x₁ e x₂. Determine o vértice da parábola e o intervalo em que f(x) < 0.
Resolução:
Δ = (−6)² − 4·1·8 = 36 − 32 = 4
x = (6 ± 2)/2 → x₁ = 2, x₂ = 4
Vértice: xᵥ = −(−6)/(2·1) = 3; yᵥ = f(3) = 9 − 18 + 8 = −1
Vértice = (3, −1). Como a = 1 > 0 (parábola abre pra cima):
f(x) < 0 para 2 < x < 4 (entre as raízes).
Questão 2 — Composição
Dadas f(x) = 2x + 1 e g(x) = x² − 3, calcule (f∘g)(2) e a inversa f⁻¹(x).
Resolução:
g(2) = 4 − 3 = 1
(f∘g)(2) = f(g(2)) = f(1) = 2(1)+1 = 3

Inversa de f: y = 2x+1 → 2x = y−1 → x = (y−1)/2
Portanto: f⁻¹(x) = (x−1)/2
Verificação: f(f⁻¹(x)) = 2·(x−1)/2 + 1 = x−1+1 = x ✓
Questão 3 — Função Exponencial
Uma população de bactérias dobra a cada hora. Começando com 500 bactérias, quantas haverá após 5 horas? Em que hora haverá mais de 16.000 bactérias?
Resolução:
Modelo: f(t) = 500 · 2^t
Após 5 horas: f(5) = 500 · 32 = 16.000 bactérias

Para f(t) > 16.000: 500 · 2^t > 16.000 → 2^t > 32 → 2^t > 2^5 → t > 5
Portanto, a partir de t = 5 horas há mais de 16.000 bactérias.
