Razões Trigonométricas no Triângulo Retângulo

As razões trigonométricas relacionam os lados de um triângulo retângulo com seus ângulos. O mnemônico SOH-CAH-TOA é a chave.

Seno — SOH

sin θ = Cateto Oposto / Hipotenusa. Varia de 0 a 1 para θ ∈ [0°, 90°]. sin 0° = 0; sin 90° = 1. Cateto oposto é o que fica "do lado de fora" do ângulo θ.

Cosseno — CAH

cos θ = Cateto Adjacente / Hipotenusa. Varia de 1 a 0 para θ ∈ [0°, 90°]. cos 0° = 1; cos 90° = 0. Cateto adjacente é o que "toca" o ângulo θ (além da hipotenusa).

Tangente — TOA

tan θ = Cateto Oposto / Cateto Adjacente = sin θ / cos θ. Não tem limite: tan 89° é muito grande; tan 90° = indefinida. tan 45° = 1 (catetos iguais).

Identidade de Pitágoras

sin²θ + cos²θ = 1 (sempre). Decorre do teorema de Pitágoras. A partir disso: sec²θ = 1 + tan²θ; csc²θ = 1 + cot²θ.

Mnemônico: SOH — CAH — TOA sin θ = Oposto / Hipotenusa cos θ = Adjacente / Hipotenusa tan θ = Oposto / Adjacente Hipotenusa = lado oposto ao ângulo reto (o maior lado) Identidade fundamental: sin²θ + cos²θ = 1
Dica UNICAP: Se sin θ = 3/5, então cos θ = 4/5 e tan θ = 3/4. Montar o triângulo retângulo de lados 3, 4, 5 (tripla pitagórica) é a forma mais rápida de calcular todas as razões.

Círculo Unitário — Visualização e Propriedades

O círculo unitário (raio = 1, centro na origem) é a ferramenta definitiva para entender trigonometria. Para qualquer ângulo θ, o ponto P = (cos θ, sin θ) está na circunferência.

Quadrantes e Sinais

Q1 (0–90°): sin>0, cos>0. Q2 (90–180°): sin>0, cos<0. Q3 (180–270°): sin<0, cos<0. Q4 (270–360°): sin<0, cos>0. Mnemônico: "Todos Sinais Começam Com" (Todos, Seno, Cosseno, Tangente).

Ângulos Relacionados

Arco suplementar (π − θ): sin(π−θ) = sin θ; cos(π−θ) = −cos θ. Arco oposto (−θ): sin(−θ) = −sin θ; cos(−θ) = cos θ. Arco complementar (π/2−θ): sin = cos do outro.

Periodicidade

Seno e cosseno: período 2π (360°). sin(θ+2π) = sin θ. Tangente: período π (180°). tan(θ+π) = tan θ. Qualquer ângulo pode ser reduzido ao intervalo [0°, 360°].

Radianos

π rad = 180°. 1 rad ≈ 57,3°. Converter: graus × π/180 = radianos. Ex: 90° = π/2; 60° = π/3; 45° = π/4; 30° = π/6. Comprimento de arco: s = r·θ (θ em rad).

Conversão: graus × π/180 = radianos radianos × 180/π = graus Tabela rápida: 30° = π/6 | 45° = π/4 | 60° = π/3 90° = π/2 | 180° = π | 270° = 3π/2 | 360° = 2π Ponto no círculo unitário: P(θ) = (cos θ, sin θ)

Funções Trigonométricas — Gráficos, Período e Amplitude

y = sin(x)

Amplitude = 1. Período = 2π. Domínio: ℝ. Imagem: [−1, 1]. Cruza o eixo x em x = nπ. Máximo em π/2; mínimo em 3π/2. Função ímpar: sin(−x) = −sin x.

y = cos(x)

Amplitude = 1. Período = 2π. Domínio: ℝ. Imagem: [−1, 1]. Cruza o eixo x em π/2 + nπ. Máximo em 0; mínimo em π. Função par: cos(−x) = cos x.

y = tan(x)

Período = π. Domínio: ℝ − {π/2 + nπ}. Imagem: ℝ. Assíntotas verticais em ±π/2, ±3π/2, etc. Função ímpar: tan(−x) = −tan x. Cresce em cada ramo.

Forma Geral y = a·sin(bx+c)+d

a = amplitude (|a|). b determina período: T = 2π/|b|. c = defasagem (deslocamento horizontal). d = deslocamento vertical. Mesma lógica vale para cosseno e tangente.

y = a · sin(bx + c) + d Amplitude: |a| Período: T = 2π/|b| (para sin/cos) ou T = π/|b| (para tan) Fase: deslocamento horizontal de −c/b Deslocamento vertical: d Exemplo: y = 3·sin(2x − π/3) + 1 Amplitude = 3 ; Período = 2π/2 = π ; Fase = π/6 para direita ; Centro = 1

Lei dos Senos e Lei dos Cossenos

Para triângulos não retângulos (oblíquos), o Pitágoras não se aplica. As Leis dos Senos e dos Cossenos generalizam as relações entre lados e ângulos.

Lei dos Senos

a/sin A = b/sin B = c/sin C = 2R. Use quando: (1) dois ângulos e um lado (AAL); (2) dois lados e o ângulo oposto a um deles (LAL-oposto). Permite encontrar qualquer lado ou ângulo faltante.

Lei dos Cossenos

c² = a²+b²−2ab·cos C. Generaliza Pitágoras. Use quando: (1) três lados (LLL) para encontrar ângulo; (2) dois lados e o ângulo entre eles (LAL-incluso). Não funciona com AAL.

Quando Usar Cada Uma

Tem 2 ângulos → Lei dos Senos. Tem 3 lados → Lei dos Cossenos. Tem 2 lados + ângulo entre eles → Lei dos Cossenos. Tem 2 lados + ângulo oposto → Lei dos Senos.

Casos Especiais

Quando C = 90°: cos 90° = 0, Lei dos Cossenos vira c² = a²+b² (Pitágoras!). Triângulo equilátero: todos os lados iguais, todos os ângulos = 60°. Lei dos Senos: a/sin 60° = 2R → R = a/√3.

Lei dos Senos: a / sin A = b / sin B = c / sin C = 2R Lei dos Cossenos: c² = a² + b² − 2ab·cos C → cos C = (a² + b² − c²) / (2ab) Notação: lados a, b, c opostos aos ângulos A, B, C respectivamente.

Identidades Trigonométricas Fundamentais
Identidade Fundamental: sin²θ + cos²θ = 1 Derivadas: 1 + tan²θ = sec²θ (divida por cos²θ) 1 + cot²θ = csc²θ (divida por sin²θ) Razões: tan θ = sin θ / cos θ cot θ = cos θ / sin θ = 1/tan θ sec θ = 1/cos θ csc θ = 1/sin θ Fórmulas de Adição: sin(A+B) = sin A·cos B + cos A·sin B sin(A−B) = sin A·cos B − cos A·sin B cos(A+B) = cos A·cos B − sin A·sin B cos(A−B) = cos A·cos B + sin A·sin B Arco Duplo: sin 2A = 2·sin A·cos A cos 2A = cos²A − sin²A = 1 − 2sin²A = 2cos²A − 1 tan 2A = 2·tan A / (1 − tan²A)
Dica UNICAP: A prova mais cobrada é: simplificar expressões como (sin²θ+cos²θ)/cos²θ = 1/cos²θ = sec²θ. Ou provar que (sin θ + cos θ)² = 1 + 2sin θ cos θ = 1 + sin 2θ. Parta sempre da identidade fundamental.

Questões-Tipo UNICAP — Resolvidas
Questão 1 — Valores Notáveis
Calcule o valor de sin 45° · cos 30° + cos 45° · sin 30°. O que essa expressão representa?
Resolução:
sin 45° = √2/2 ; cos 30° = √3/2 ; cos 45° = √2/2 ; sin 30° = 1/2
Expressão = (√2/2)(√3/2) + (√2/2)(1/2) = √6/4 + √2/4 = (√6+√2)/4
Essa é a fórmula de sin(A+B) com A=45° e B=30°: sin 75° = (√6+√2)/4
Questão 2 — Lei dos Cossenos
Um triângulo tem lados a=5, b=7 e c=8. Encontre o cosseno e o valor aproximado do ângulo C (oposto ao lado c=8).
Resolução:
cos C = (a²+b²−c²)/(2ab) = (25+49−64)/(2·5·7) = 10/70 = 1/7 ≈ 0,143
C = arccos(1/7) ≈ 81,8° — ângulo obtuso? Não! cos>0, então C é agudo.
cos C = 1/7, C ≈ 81,8°
Questão 3 — Identidade
Simplifique a expressão: (1 − sin²θ) / cos θ + sin²θ / cos θ
Resolução:
= [1 − sin²θ + sin²θ] / cos θ (mesmo denominador)
= 1 / cos θ
= sec θ
(Usamos: 1 − sin²θ = cos²θ, mas como os sin²θ se cancelam antes, fica direto = 1/cosθ)
